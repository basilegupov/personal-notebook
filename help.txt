        Довідка для Контактного Асистента
=======================================================

Вітаємо у Контактному Асистенті! Ця програма допомагає ефективно керувати вашими контактами.

Список доступних команд:
- `hello`: Привітати бота.
- `add name phone=<номер_телефону> email=<адреса_електронної_пошти> birthday=<РРРР-ММ-ДД> address=<адреса>`: Додати новий контакт.
  - Приклад: `add Іван Петров phone=1234567890 email=ivan@example.com birthday=1990-01-01 address=вул. Головна, 1`
  - Примітка: Поля `phone`, `email`, `birthday`, та `address` є опціональними, і ви можете вказати тільки ті, які необхідно.
  - Приклад неправильного введення: `add Іван Петров phone=1234 email=ivan@example.com birthday=1990-01-01 address=вул. 
    Головна, 1` (невірний номер телефону).
- `change name <ім'я_контакту> <поле_для_зміни>=<нове_значення>`: Змінити деталі контакту.
  - Приклад: `change Іван Петров email=newemail@example.com`
- `phone name <ім'я_контакту>`: Отримати номер телефону контакту.
  - Приклад: `phone Іван Петров`
- `search <запит>`: Пошук контактів за ім'ям, номером телефону, електронною поштою чи адресою.
  - Приклад: `search Іван`
- `email <ім'я_контакту або email>`: Отримати адресу електронної пошти контакту.
  - Приклад: `email ivan@example.com`
- `show`: Показати всі контакти в телефонній книзі.
  - Приклад: `show`
- `birthday <дні>`: Показати найближчі дні народження протягом вказаної кількості днів.
  - Приклад: `birthday 7`
- `close`, `exit`, `good bye`: Вийти з програми.
  - Приклад: `close`, `exit`, `good bye`
- `sorted <шлях_до_папки>`: Впорядкувати файли у вказаній папці.
  - Приклад: `sorted /шлях/до/папки`
- `notes`: Відкрити функціонал заміток.
  - Приклад: `notes`

    Вітаємо у Контактному Асистенті! Асистент допомагає ефективно керувати вашими нотатками.

    Список доступних команд:
    - `1`: Додати нову нотатку.
       - Приклад: Введіть `1` для додавання нової нотатки та слідуйте інструкціям.
    - `2`: Пошук нотаток за заголовком чи змістом.
       - Приклад: Введіть `2` та введіть запит для пошуку (заголовок чи зміст).
    - `3`: Редагувати існуючу нотатку.
       - Приклад: Введіть `3` для редагування існуючої нотатки та слідуйте інструкціям.
    - `4`: Видалити нотатку.
      - Приклад: Введіть `4` для видалення нотатки та слідуйте інструкціям.
    - `5`: Вивести всі нотатки в консоль.
       - Приклад: Введіть `5` для виведення всіх нотаток в консоль.
    - `6`: Вивести нотатку за заголовком.
       - Приклад: Введіть `6` для виведення конкретної нотатки за заголовком.
    - `7`: Вивести нотатку(-и) за тегом.
       - Приклад: Введіть `7` для виведення нотаток за певним тегом.
    - `8`: Вивести нотатки за певними тегами у вигляді таблиці.
       - Приклад: Введіть `8` для виведення нотаток за певними тегами у вигляді таблиці.
    - `9`: Вийти з програми.
       - Приклад: Введіть `9` для виходу з програми.       

Примітка: Використовуйте подвійні лапки для значень, що містять пробіли.

Бажаємо вам приємного користування Контактним Асистентом!
